# Iozen University (demo)

A playful fake university landing page inspired by [iozen.ai](https://iozen.ai) — same warm palette, DM Sans typography, glassy nav, and conversational copy — but for a totally made-up campus.

## Run locally

```bash
# from this folder
python3 -m http.server 8080
# open http://localhost:8080
```

Or open `index.html` directly in a browser.

## Deploy

Static files only — push to GitHub and enable Pages, or host anywhere.

## Not official

This is a fan demo. [ioZen](https://iozen.ai) is the real product.
